图片压缩方法教程

为什么需要压缩图片？

高像素手机拍出的照片通常在 3-10MB，上传到网站、发送邮件、上传证件照时经常遇到文件大小限制。压缩图片可以：

  • 加快网页加载速度（影响 SEO 排名）
  • 节省手机存储空间
  • 满足各平台上传大小限制
  • 减少流量消耗

方法一：在线工具压缩（最简单）

最推荐的方式，不需要安装任何软件：

  1. 打开 在线图片压缩工具
  2. 拖拽或点击上传图片（支持批量）
  3. 调整质量滑块（推荐 70%-85%）
  4. 点击"压缩并下载"

优点：免费、无需安装、支持批量、隐私安全（不上传服务器）

方法二：手机自带功能

iPhone：设置 → 相机 → 格式 → 选择"高效"（HEIC格式自动压缩约50%）

Android：相机设置 → 选择"中等"或"低"画质

方法三：电脑软件

  • TinyPNG — 免费在线压缩，支持 PNG/JPG
  • Photoshop — 导出为 Web 格式，调整品质参数
  • macOS 预览 — 文件 → 导出 → 调整 JPEG 品质
