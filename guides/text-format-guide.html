中文排版规范

中文排版基本规范

首行缩进

中文段落首行应缩进2个全角空格（2em）：

  • Word中：段落设置 → 首行缩进 2字符
  • 网页中：CSS `text-indent: 2em`
  • Markdown中：段落开头输入两个全角空格（U+3000）

行间距

中文推荐行间距为字号的1.5-2倍，例如16px字号配合24px行高。正文推荐使用 `line-height: 1.8`。

全角 vs 半角

类型适用示例
全角中文标点、特殊符号，。！？：；（）【】""
半角英文标点、数字、字母, . ! ? : ; ( ) [ ] " "

中英文混排技巧

  1. 中英文之间加空格：排版更美观，阅读更舒适
  2. 中文用全角标点：逗号用"，"而不是","
  3. 英文用半角标点：逗号用","而不是"，"
  4. 避免连续全角空格：多余空格影响排版效果
  5. 统一数字格式：一般使用半角阿拉伯数字

Markdown 排版技巧

# 一级标题

常见文本排版问题

  • 多余空行：从网页复制文本常带多余空行，需要清理
  • 首尾空格：每行文字前后多余空格影响对齐
  • 标点错误：英文标点混入中文文本
  • 编码问题：全角/半角混用导致排版混乱
