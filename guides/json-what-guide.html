JSON 是什么？

JSON（JavaScript Object Notation）是当今互联网最常用的数据交换格式。

JSON 基本语法

{
  "name": "张三",
  "age": 25,
  "skills": ["JavaScript", "Python"],
  "active": true
}

6 种数据类型

  • 对象 {} — 键值对集合，类似字典/Map
  • 数组 [] — 有序值列表
  • 字符串 "" — 用双引号包裹的文本
  • 数字 — 整数或浮点数
  • 布尔值 — true 或 false
  • null — 空值
